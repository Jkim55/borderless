About this App

An app to aggregate info for smart travel planning. Information targeted for display include visa requirements, travel advisories(country and neighboring countries), vaccination recommendation, electricity voltage info, telephone pre-fixes, currency exchange rates as well as recent news stories related to that country. Someone who is interested in having a reference for organizing details on a trip or is prepping final details for a trip that's already been planned while staying informed on current events.

API's consumed: - Travelbriefing.org public API - Google Maps API - Country.io Data - news service - Yandex Translate
